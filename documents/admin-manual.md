## 1. 관리 개요

성당 홈페이지는 세 가지 콘텐츠로 구성됩니다:

| 콘텐츠 | 관리 방법 | 도구 |
|--------|----------|------|
| **공지사항** | 관리자 패널 (웹 페이지) | 웹 브라우저 |
| **주보** | Google Drive에 파일 업로드 | Google Drive |
| **앨범** | Google Drive에 폴더 + 사진 업로드 | Google Drive |

### 반영 시간

콘텐츠를 추가하거나 수정하면 **약 5~20분 후** 홈페이지에 자동 반영됩니다.
(시스템이 5분마다 변경사항을 확인하고, 변경이 감지되면 홈페이지를 자동 업데이트합니다.)

### 필요한 것

- Google 계정 (성당 Google Drive 접근 권한)
- 관리자 비밀번호 (공지사항 관리용)
- 웹 브라우저 (Chrome 권장)

## 2. 공지사항 관리

공지사항은 **관리자 패널**에서 작성, 수정, 삭제합니다.

### 2.1 관리자 패널 접속

**방법 1 — 홈페이지에서 접속 (권장)**

1. 웹 브라우저에서 아래 주소로 접속합니다:
   ```
   https://communications-ct.github.io/ctkorean/admin
   ```
2. 관리자 패널이 화면에 표시됩니다.

**방법 2 — 직접 접속**

로그인 상태에서 접속이 안 될 경우, **시크릿(비공개) 모드**를 사용합니다:

- **Chrome**: `Ctrl + Shift + N` (Mac: `Cmd + Shift + N`)
- **Safari**: `파일` → `새로운 개인 정보 보호 윈도우`
- **Edge**: `Ctrl + Shift + N`

시크릿 창에서 아래 주소로 접속합니다:
```
https://script.google.com/macros/s/AKfycbysgcyhqJWBNeUrIRBNwT7vqU2dEQc-ZzE4YNYZtjSf8vddUFxr9TrWVI2DU1D6Dpxa-g/exec?mode=admin
```

### 2.2 로그인

1. 관리자 패널에 접속하면 **로그인 화면**이 나타납니다.
2. **관리자 비밀번호**를 입력합니다.
3. **로그인** 버튼을 클릭합니다.

> 비밀번호를 모르는 경우 성당 사무실에 문의하세요.

### 2.3 새 공지사항 작성

1. 로그인 후 **공지사항 목록** 화면이 나타납니다.
2. 오른쪽 상단의 **+ 새 공지** 버튼을 클릭합니다.
3. 아래 항목을 입력합니다:

| 항목 | 설명 | 필수 |
|------|------|------|
| **날짜** | 공지 날짜 (달력에서 선택) | 필수 |
| **제목** | 공지사항 제목 | 필수 |
| **내용** | 공지사항 본문 (줄바꿈 가능) | 선택 |
| **링크** | 관련 외부 링크 (예: YouTube) | 선택 |
| **긴급 공지** | 체크하면 홈페이지 상단에 빨간 배너로 표시 | 선택 |
| **이미지 첨부** | 공지에 사진 첨부 (10MB 이하, 여러 장 가능) | 선택 |

4. 모든 내용을 입력한 후 **저장** 버튼을 클릭합니다.
5. "공지사항이 등록되었습니다" 메시지가 나타나면 성공입니다.

### 2.4 공지사항 수정

1. 공지사항 목록에서 수정할 공지의 **수정** 버튼을 클릭합니다.
2. 내용을 수정합니다.
3. **저장** 버튼을 클릭합니다.

### 2.5 공지사항 삭제

1. 공지사항 목록에서 삭제할 공지의 **삭제** 버튼을 클릭합니다.
2. 확인 팝업이 나타나면 **확인**을 클릭합니다.

> ⚠️ 삭제된 공지사항은 복구할 수 없습니다. 신중하게 삭제하세요.

### 2.6 이미지 첨부하기

1. 공지사항 작성/수정 화면에서 **이미지 첨부** 항목의 **파일 선택**을 클릭합니다.
2. 첨부할 이미지 파일을 선택합니다 (여러 장 선택 가능).
3. 선택한 이미지의 미리보기가 표시됩니다.
4. 이미지를 제거하려면 미리보기의 **X** 버튼을 클릭합니다.

> 이미지 제한: 1장당 최대 10MB, JPG/PNG/GIF 형식 지원

### 2.7 로그아웃

1. 오른쪽 상단의 **로그아웃** 버튼을 클릭합니다.
2. 또는 브라우저 탭을 닫으면 자동으로 로그아웃됩니다.

## 3. 주보 관리

주보는 **Google Drive**에 파일을 올리면 자동으로 홈페이지에 반영됩니다.

### 3.1 주보 폴더 접속

1. 웹 브라우저에서 Google Drive에 접속합니다: https://drive.google.com
2. 성당 계정으로 로그인합니다.
3. **주보 폴더**를 찾아 들어갑니다.

   또는 아래 링크로 직접 접속합니다:
   ```
   https://drive.google.com/drive/folders/1rkKcIU165hbu-ovgbjVKKKijAZI9LXpq
   ```

### 3.2 주보 업로드

1. 주보 폴더에 들어갑니다.
2. 주보 파일(PDF 또는 이미지)을 **드래그 앤 드롭**하거나, **+ 새로 만들기** → **파일 업로드**를 클릭합니다.

### 3.3 파일 이름 규칙 (중요!)

파일 이름은 반드시 **날짜로 시작**해야 합니다. 날짜가 있어야 홈페이지에서 올바른 순서로 표시됩니다.

**올바른 예시:**
```
2026-03-02 주보.pdf
2026-02-23 주보.pdf
2026.03.02 사순절 주보.pdf
```

**잘못된 예시:**
```
주보.pdf           ← 날짜 없음
3월 주보.pdf        ← 날짜 형식이 아님
```

> 파일 형식: PDF 파일을 권장합니다. 이미지(JPG, PNG)도 가능합니다.

### 3.4 주보 삭제

1. Google Drive 주보 폴더에서 해당 파일을 **우클릭** → **삭제**합니다.
2. 약 5~20분 후 홈페이지에서도 제거됩니다.

### 3.5 주보 확인

업로드 후 홈페이지 반영 전에 데이터를 바로 확인하려면:
```
https://script.google.com/macros/s/AKfycbysgcyhqJWBNeUrIRBNwT7vqU2dEQc-ZzE4YNYZtjSf8vddUFxr9TrWVI2DU1D6Dpxa-g/exec?type=bulletin
```
이 주소로 접속하면 현재 등록된 주보 목록이 JSON 형태로 표시됩니다.

## 4. 앨범 관리

앨범은 **Google Drive**에 폴더를 만들고 사진을 올리면 자동으로 홈페이지에 반영됩니다.

### 4.1 앨범 폴더 접속

1. 웹 브라우저에서 Google Drive에 접속합니다: https://drive.google.com
2. 성당 계정으로 로그인합니다.
3. **앨범 폴더**를 찾아 들어갑니다.

   또는 아래 링크로 직접 접속합니다:
   ```
   https://drive.google.com/drive/folders/1TCbyyWEBZr3Tifid8yhqMg7F-FbQkX3X
   ```

### 4.2 새 앨범 만들기

**단계 1 — 폴더 만들기**

1. 앨범 폴더 안에서 **+ 새로 만들기** → **새 폴더**를 클릭합니다.
2. 폴더 이름을 입력합니다.

**폴더 이름 규칙 (중요!):**

폴더 이름은 반드시 **날짜로 시작**해야 합니다.

```
2026-03-02 사순절 행사
2026-02-16 성당 설날 잔치
2026.01.05 신년미사
```

**단계 2 — 사진 올리기**

1. 방금 만든 폴더를 더블클릭하여 들어갑니다.
2. 사진 파일들을 **드래그 앤 드롭**하거나, **+ 새로 만들기** → **파일 업로드**를 클릭합니다.
3. 여러 장을 한꺼번에 업로드할 수 있습니다.

> **썸네일**: 폴더 안의 **첫 번째 이미지**가 자동으로 홈페이지의 앨범 대표 이미지(썸네일)가 됩니다.

### 4.3 앨범에 사진 추가

1. 앨범 폴더 안의 해당 앨범 폴더를 엽니다.
2. 추가할 사진을 업로드합니다.

### 4.4 앨범 삭제

1. 앨범 폴더 안에서 삭제할 앨범 폴더를 **우클릭** → **삭제**합니다.
2. 약 5~20분 후 홈페이지에서도 제거됩니다.

> ⚠️ 폴더를 삭제하면 안의 모든 사진도 함께 삭제됩니다.

### 4.5 앨범 확인

업로드 후 홈페이지 반영 전에 데이터를 바로 확인하려면:
```
https://script.google.com/macros/s/AKfycbysgcyhqJWBNeUrIRBNwT7vqU2dEQc-ZzE4YNYZtjSf8vddUFxr9TrWVI2DU1D6Dpxa-g/exec?type=albums
```
이 주소로 접속하면 현재 등록된 앨범 목록이 JSON 형태로 표시됩니다.

## 5. 홈페이지 반영 확인

### 5.1 자동 반영 (기본)

콘텐츠를 추가/수정/삭제하면 시스템이 **5분마다** 변경을 감지하고, 변경이 있으면 자동으로 홈페이지를 업데이트합니다.

**전체 과정:**
```
콘텐츠 변경 (공지/주보/앨범)
  ↓ (최대 5분)
시스템이 변경 감지
  ↓ (약 1~2분)
홈페이지 자동 빌드 및 배포
  ↓
홈페이지에 반영 완료 (총 5~20분)
```

### 5.2 수동 즉시 반영

급하게 반영해야 할 경우, GitHub에서 수동으로 빌드를 실행할 수 있습니다:

1. 아래 주소로 접속합니다:
   ```
   https://github.com/communications-ct/ctkorean/actions
   ```
2. 왼쪽 메뉴에서 **Deploy Astro to GitHub Pages**를 클릭합니다.
3. 오른쪽의 **Run workflow** 버튼을 클릭합니다.
4. **Run workflow**를 한 번 더 클릭합니다.
5. 약 1~2분 후 홈페이지에 반영됩니다.

### 5.3 반영 확인

- **한국어 홈페이지**: https://communications-ct.github.io/ctkorean/ko/
- **영어 홈페이지**: https://communications-ct.github.io/ctkorean/en/
- **공동체 소식 (한국어)**: https://communications-ct.github.io/ctkorean/ko/news
- **공동체 소식 (영어)**: https://communications-ct.github.io/ctkorean/en/news

> 변경이 보이지 않으면 브라우저에서 **Ctrl + Shift + R** (Mac: **Cmd + Shift + R**)로 강력 새로고침하세요.

## 6. 긴급 공지 사용법

긴급 공지를 등록하면 홈페이지 **최상단에 빨간 배너**가 표시되어 방문자에게 즉시 알립니다.

### 긴급 공지 등록

1. 관리자 패널에서 공지사항 작성 시 **긴급 공지** 체크박스를 체크합니다.
2. 저장합니다.

### 긴급 배너 표시 기간

- 공지 **날짜로부터 7일간** 홈페이지 상단에 빨간 배너가 표시됩니다.
- 7일 후 다음 홈페이지 빌드 시 자동으로 배너가 사라집니다.
- 공지사항 목록에서는 계속 빨간 "긴급" 배지가 표시됩니다.

### 사용 예시

- 미사 시간 변경
- 미사 취소
- 긴급 행사 안내
- 천재지변 관련 공지

> ⚠️ 긴급 공지는 정말 긴급한 경우에만 사용하세요. 자주 사용하면 효과가 떨어집니다.

## 7. 자주 묻는 질문 (FAQ)

### Q. 관리자 비밀번호를 잊어버렸어요.

Google Apps Script의 프로젝트 설정에서 확인할 수 있습니다. 스크립트 편집 권한이 있는 관리자에게 문의하세요.

### Q. 공지사항을 올렸는데 홈페이지에 안 보여요.

1. **5~20분 기다려 주세요.** 자동 반영에 시간이 필요합니다.
2. 급하면 [5.2 수동 즉시 반영](#52-수동-즉시-반영)을 참고하세요.
3. 브라우저 캐시 문제일 수 있습니다. **Ctrl + Shift + R**로 강력 새로고침하세요.

### Q. 주보를 올렸는데 목록에 안 나와요.

1. 파일 이름이 **날짜로 시작**하는지 확인하세요. (예: `2026-03-02 주보.pdf`)
2. 올바른 폴더에 올렸는지 확인하세요.
3. 5~20분 기다려 주세요.

### Q. 앨범 사진을 올렸는데 썸네일이 이상해요.

앨범 폴더의 **첫 번째 이미지**가 썸네일이 됩니다. 대표 이미지로 쓸 사진의 파일명을 `00_` 으로 시작하면 첫 번째가 됩니다.

### Q. 관리자 패널에 로그인이 안 돼요.

1. 비밀번호가 정확한지 확인하세요 (대소문자, 특수문자 구분).
2. **시크릿(비공개) 창**에서 접속해 보세요.

### Q. 이미지가 업로드 안 돼요.

1. 파일 크기가 **10MB 이하**인지 확인하세요.
2. 파일 형식이 **JPG, PNG, GIF**인지 확인하세요.
3. Google Drive 저장 용량이 남아 있는지 확인하세요.

### Q. Google Drive 폴더 링크를 잃어버렸어요.

| 폴더 | 링크 |
|------|------|
| 주보 | https://drive.google.com/drive/folders/1rkKcIU165hbu-ovgbjVKKKijAZI9LXpq |
| 앨범 | https://drive.google.com/drive/folders/1TCbyyWEBZr3Tifid8yhqMg7F-FbQkX3X |
| 공지사항 이미지 | https://drive.google.com/drive/folders/1kxntITty41lImTQs8nw_ddT2OSU34urb |

### Q. 실수로 공지사항을 삭제했어요.

삭제된 공지사항은 복구할 수 없습니다. 같은 내용으로 새 공지사항을 작성해 주세요.

### Q. 실수로 주보/앨범 파일을 삭제했어요.

Google Drive 휴지통에서 30일 이내에 복구할 수 있습니다:
1. Google Drive에서 왼쪽 메뉴의 **휴지통**을 클릭합니다.
2. 삭제한 파일을 **우클릭** → **복원**합니다.
